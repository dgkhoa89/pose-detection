## Installation
### Install python package
`pip install -r setup.txt`

If the command does not work, please install these following package
- open-cv
- mediapipe
- numpy
- keras
- streamlit_webrtc

## Start application
`streamlit run .\inference_test.py`

## Model
- Currently model can recognize only: hello, thank
- We're adding more
